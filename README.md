# Change Detection Assignment

## How to Run
```bash
python3 changeDetection.py \
    --input_folder "data/person1" \
    --output_folder "results/person1" \
    --input_ext "png" \
    --output_ext "png" \
    --video_format "mp4"

Requirements

Python 3.8+ , numpy, opencv-python ,matplotlib

Install with:

pip install numpy opencv-python matplotlib
